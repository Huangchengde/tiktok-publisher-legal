Authorization takes place on TikTok. KnowGrow does not ask for or store your TikTok password.

Revoke access

You can revoke an application's authorization from your TikTok account security settings. Revocation prevents future API actions.

Report a problem

For general support, report a reproducible issue in the public project repository. Never include passwords, access tokens, private videos, or personal data.

Open a support issue →

Before contacting support

Include the page URL, browser name, approximate time, and a description of what you expected. Do not attach private media or TikTok credentials.

Service availability

Video preparation can be completed locally in the browser. TikTok authorization and posting availability depend on TikTok APIs, account eligibility, regional availability, and TikTok platform status.
